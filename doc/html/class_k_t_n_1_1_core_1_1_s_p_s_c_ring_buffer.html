Detailed Description

template<typename T>
class KTN::Core::SPSCRingBuffer< T >

A single-producer, single-consumer ring buffer implementation.

This class provides a thread-safe ring buffer that can be used in a single-producer, single-consumer scenario. It supports enqueueing and dequeueing elements, as well as checking the size and emptiness of the buffer.

Template Parameters
TThe type of elements stored in the ring buffer.
