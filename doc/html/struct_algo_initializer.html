Detailed Description

The AlgoInitializer struct represents the parameters required to initialize the algorithm.

the idea is to create an easy to use all-in-one struct to init the algo with std::string name, std::string guid, int algoid, ExchangeSender& ordsender, vector<int> channels, int cpu, CmeSecMaster & sec
