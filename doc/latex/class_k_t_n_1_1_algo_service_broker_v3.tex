\doxysubsection{Detailed Description}
The \doxylink{class_k_t_n_1_1_algo_service_broker_v3}{Algo\+Service\+Broker\+V3} class routes messages to and from child instances of \doxylink{class_k_t_n_1_1_algo_base_v3}{Algo\+Base\+V3}. 

This class provides functionality for initializing Zero\+MQ, printing information, stopping all algorithms, resetting positions, sending commands, and handling algorithm commands and orders.
