\doxysubsection{Detailed Description}
Struct representing the risk parameters for an algorithm. 

This struct is used to represent the risk parameters for an algorithm. The model we use here is called a "{}\+Triple Barrier"{} risk model, where we have a profit taking value, a stop loss value, and a time to live value.
